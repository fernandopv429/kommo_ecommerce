# Nexus.ai - Plataforma de Automação

Projeto focado em automação com Inteligência Artificial para Clínicas e E-Commerces. Transformamos operações completas em sistemas eficientes de conversão, atendimento e retenção usando IA.

## Como Iniciar

Este projeto é uma aplicação Single Page (SPA) React utilizando Vite.

Para rodar localmente:

1. Instale as dependências:
```bash
npm install
```

2. Execute o servidor de desenvolvimento:
```bash
npm run dev
```

3. Acesse a aplicação na porta informada pelo Vite.
